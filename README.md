# 뉴스 모니터링 시스템 (news_curating)

AI·금융 뉴스를 **카테고리별 중요도 기준으로 자동 수집·채점·중복제거**해서,
의미 있는 뉴스만 골라 대시보드에 보여주는 모니터링 시스템입니다.

> **핵심 원칙 — "빈도 최소화, 관심도 최대화."**
> 글로벌은 판도 전환급 뉴스만, 국내·우리금융은 전용 트랙으로 촘촘히 챙깁니다.

🔗 **라이브 대시보드 → https://news-curating.vercel.app**

---

## 무엇을 하나요

- 여러 뉴스 소스(RSS·토스인베스트 종목 뉴스 등)에서 AI·금융 뉴스를 **자동 수집**합니다.
- 각 기사를 카테고리 기준에 따라 **중요도 0~10점으로 채점**하고, 임계값을 넘는 것만 "중요뉴스"로 추립니다.
- 같은 사건을 다룬 **중복 기사는 묶어 하나만** 보여줍니다.
- 대시보드에서 **개요 / 중요뉴스 / 전체뉴스 / 카테고리별**로 나눠 볼 수 있고, 기사별로 평가(피드백)를 남길 수 있습니다.

## 카테고리

| 카테고리 | 설명 |
|---|---|
| 글로벌 AI | 해외 AI 핵심 동향 (판도 전환급만) |
| 글로벌 금융 AI | 해외 금융권 AI 뉴스 |
| 국내 금융 AI | 국내 금융지주·은행의 AI 뉴스 |
| 우리금융그룹 | 우리금융 전용 트랙 |

## 기술 스택

- **백엔드:** Python · FastAPI
- **프론트엔드:** 바닐라 HTML/CSS/JS (단일 페이지 대시보드)
- **저장소:** SQLite (로컬) / Turso (클라우드)
- **배포:** Vercel + Turso (영구 URL)
